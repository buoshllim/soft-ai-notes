# "AI야 나를 알려줘"

2026 HR AX PLAN의 일환으로 기획한 AI 자기소개 캠페인 프롬프트임.  
자주 쓰는 AI에 아래 프롬프트를 입력하면 됨.

---

## 프롬프트

```
Part 1. 지금까지 함께 나눈 대화와 작업을 돌아보고 아래 항목별로 정리해줘.

1. 내가 주로 다루는 업무 영역과 실제 역할 및 책임 범위
2. 내가 자주 고민하는 문제 유형과 주요 판단 및 의사결정 사례
3. 내 사고 방식과 커뮤니케이션 스타일
4. 강점으로 보이는 역량
5. 자주 도움을 요청하는 영역


Part 2. 아래 질문을 한 개씩 물어봐주고 내 답변을 정리해서 요약해줘.

1. 5~10년 후 내가 지향하는 커리어 방향은? (전문가/리더/경영자/미정)
2. 향후 경험해보고 싶은 역할이나 업무 영역은?
3. 가장 효과적이었던 성장방식은? (도전과제/역할확장/멘토링/코칭/교육)
4. 지금까지 경력 전체에서 가장 중요했던 경험은?
5. 지금보다 더 큰 책임을 맡는 것에 대한 나의 생각은? (희망함/시기상조)
```

---

AI를 오래 써온 사람일수록 Part 1의 결과가 더 풍부하게 나옴.  
Part 2는 AI가 한 질문씩 순서대로 물어보는 방식으로 진행됨.
